Dropbox Forum Extender+

Just a little script to beef up the forums, and add functionality to the forums.

The Gist

This script adds many things to the forums, including post and thread highlighting, theming, and a message system. This can greatly increase productivity and speed when handling forum replies, which is especially good for forum Super Users and regulars.

The Backstory

The script started as a joke between the Super Users. It all started in this forum thread, and we've always said how it would be cool if we had icons, and this script did exactly that: added icons. Animated Nyan Cat icons to be exact.
